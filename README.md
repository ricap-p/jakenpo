# 🎮 JakenPô - DevClub vs Computer

Projeto de jogo **Pedra, Papel e Tesoura (JakenPô)** desenvolvido com **HTML, CSS e JavaScript**, com foco em prática de lógica, manipulação do DOM e interação com o usuário.

---

## 🚀 Funcionalidades

* 🎮 Escolha entre Pedra, Papel ou Tesoura
* 🤖 Jogada automática do computador
* ⚖️ Comparação das jogadas
* 📊 Exibição do resultado em tempo real
* 🎨 Interface moderna com estilo minimalista

---

## 🧠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 💻 Como funciona

O jogador escolhe uma opção clicando em um dos botões:

* ✊ Pedra
* ✋ Papel
* ✌️ Tesoura

O computador gera uma escolha aleatória.

O sistema compara as jogadas e exibe o resultado na tela:

* 🟢 Vitória
* 🔴 Derrota
* ⚪ Empate
